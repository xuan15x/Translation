## 已知限制

1. **性能监控**: 依赖 `psutil` 库，需要安装
   ```bash
   pip install psutil
   ```

2. **翻译执行**: 需要有效的API密钥配置
   - 检查 `config/config.json` 中的API密钥
   - 确保API提供商可用

3. **历史记录**: 需要SQLite数据库支持
   - 自动创建 `data/terminology.db`
   - 首次运行可能为空

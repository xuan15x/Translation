# 📜 历史记录功能使用指南

## ✅ 功能已完善

历史记录功能已经完全实现，现在可以在 GUI 中查看、搜索和导出翻译历史记录。

## 🎯 主要功能

### 1. 查看历史记录
- 点击主界面的 **📜 查看历史** 按钮
- 显示最近的翻译记录（默认 100 条）
- 以表格形式展示：时间、原文、译文、语言、状态、模型

### 2. 搜索历史记录
- 在搜索框中输入关键词
- 点击 **搜索** 按钮
- 支持搜索原文和译文内容
- 点击 **刷新** 按钮重新加载所有记录

### 3. 查看详情
- 双击任意历史记录
- 弹出详细信息窗口
- 显示完整的翻译信息

### 4. 导出历史记录
- 点击 **📤 导出 Excel** 按钮
- 选择保存位置
- 导出为 Excel 文件（.xlsx 格式）
- 文件名格式：`translation_history_YYYYMMDD_HHMMSS.xlsx`

### 5. 统计信息
窗口顶部显示统计信息：
- **总记录数**：所有翻译记录总数
- **成功数**：翻译成功的记录数
- **失败数**：翻译失败的记录数
- **成功率**：成功记录占比（百分比）
- **最近批次**：最近一次翻译的批次 ID
- **最后更新**：最近一次翻译的时间

## 🔧 技术实现

### 后端支持
- **模块**：`service/translation_history.py`
- **数据库**：SQLite (`translation_history.db`)
- **主要功能**：
  - 自动记录所有翻译历史
  - 支持多条件查询
  - 统计信息计算
  - Excel 导出功能

### GUI 实现
- **模块**：`presentation/modern_gui_app.py`
- **主要方法**：
  - `_show_history_window()`：显示历史记录窗口
  - `_load_history_list()`：加载历史数据
  - `_update_stats()`：更新统计信息
  - `_search_history()`：搜索历史记录
  - `_show_record_detail()`：显示记录详情
  - `_export_history()`：导出历史记录

## 📊 数据字段

每条历史记录包含以下字段：

| 字段 | 说明 |
|------|------|
| 时间 | 翻译完成时间 |
| 原文 | 源语言文本 |
| 译文 | 翻译后的文本 |
| 语言 | 目标语言 |
| 状态 | 成功/失败 |
| 模型 | 使用的 AI 模型 |
| API 提供商 | 使用的 API 服务 |
| 批次 ID | 所属翻译批次 |
| 源文件 | 来源文件路径 |

4. **问题追溯**：查看失败记录，分析原因

## 🔍 高级查询

支持以下查询条件（代码层面）：
- 关键词搜索（原文/译文）
- 目标语言过滤
- 状态过滤（成功/失败）
- 日期范围查询
- 批次 ID 查询

示例（编程使用）：
```python
from service.translation_history import get_history_manager

manager = get_history_manager()

# 搜索包含特定关键词的记录
records = manager.search_records(keyword='游戏', limit=50)

# 查询特定日期的记录
records = manager.search_records(
    start_date='2026-01-01',
    end_date='2026-12-31',
    limit=100
)

# 只看法语翻译
records = manager.search_records(target_lang='法语', limit=50)
```

## 📝 注意事项

1. **数据库位置**：`translation_history.db` 在项目根目录
2. **自动记录**：所有翻译操作会自动记录，无需手动操作
3. **性能优化**：默认加载最近 100 条，避免界面卡顿
4. **数据备份**：建议定期导出历史记录

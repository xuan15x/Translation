## 💡 使用建议

### 新手用户（推荐配置）

**只启用 T1 语言：**
```yaml
target_languages:
  - 英语
  - 日语
  - 韩语
  - 法语
  - 德语
  - 西班牙语
```

### 进阶用户（推荐配置）

**启用 T1 + T2 语言：**
```yaml
target_languages:
  # T1
  - 英语
  - 日语
  - 韩语
  - 法语
  - 德语
  - 西班牙语
  - 意大利语
  - 葡萄牙语
  - 俄语
  
  # T2
  - 泰语
  - 越南语
  - 阿拉伯语
  - 土耳其语
  - 印地语
```

### 专业用户（全量配置）

**启用所有语言：**
```yaml
target_languages:
  # 直接使用 config.py 中的 TARGET_LANGUAGES
  # 包含 T1 + T2 + T3 全部 118+ 种语言
```

## 🔧 配置方式

### 方式 1：在配置文件中设置

编辑 `config/config.yaml`：
```yaml
target_languages:
  - 英语
  - 日语
  - 韩语
  # ... 根据需要添加
```

### 方式 2：在 GUI 界面选择

启动程序后，在语言选择区域勾选需要的语言。

### 方式 3：通过代码访问

```python
from config.config import TARGET_LANGUAGES, LANGUAGES_BY_TIER, LANGUAGES_BY_CONTINENT

# 获取所有语言
all_langs = TARGET_LANGUAGES

# 按优先级获取
t1_langs = LANGUAGES_BY_TIER['T1']
t2_langs = LANGUAGES_BY_TIER['T2']
t3_langs = LANGUAGES_BY_TIER['T3']

# 按大洲获取
asian_langs = LANGUAGES_BY_CONTINENT['亚洲']
european_langs = LANGUAGES_BY_CONTINENT['欧洲']
```

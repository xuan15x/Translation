## 🔧 技术实现

### 锚点 ID 生成规则
所有目录链接遵循以下规则生成锚点 ID：

1. **转为小写**: 将标题转换为小写字母
2. **空格转横杠**: 所有空格替换为 `-`
3. **移除特殊字符**: 移除 emoji 和其他非字母数字字符
4. **连续横杠合并**: 多个连续的 `-` 合并为一个

**示例**:
```
标题：## ✨ 核心特性
锚点：#-核心特性

标题：### 系统层次架构（六层设计）
锚点：#系统层次架构六层设计

标题：## 🚀 快速开始
锚点：#-快速开始
```

### 目录格式规范
```markdown
## 📑 目录

- [一级标题](#锚点-id)
  - [二级标题](#锚点-id)
    - [三级标题](#锚点-id)
      - [四级标题](#锚点-id)

---
```
